# Semantic Prompt Search Assistant with Notion, ChatGPT, and HuggingFace

Advanced n8n automation for Semantic Prompt Search Assistant with Notion, ChatGPT, and HuggingFace.

## Overview
- Category: Internal Wiki, AI RAG
- Complexity: advanced
- Source: n8n workflow template export

## What This Automation Does
Boost ChatGPT with Notion's prompt library for smarter replies using semantic search and automatic embeddings. Fast setuplearn more now!

## Included Files
- `workflow.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.
